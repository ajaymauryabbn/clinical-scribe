# Clinical Scribe

AI-powered clinical documentation for Indian doctors.

This repository contains the backend and frontend for the Clinical Scribe application.

## Deployment

The backend is deployed on Render, and this README serves as the entry point for Hugging Face Spaces using the Docker SDK.

### Hugging Face Setup
This space is configured to run the FastAPI backend.
